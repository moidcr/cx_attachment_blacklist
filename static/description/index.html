Blacklist Attachments

Get rid of useless files in "Attachment(s)" menu!

Before
After
Sometimes you receive a lot on unwanted attachments (e.g. logos, banners, social network icons etc) that are attached to the message alongside with a really valuable files. Odoo stores such attachments and displays them in "Attachment(s)" dropdown menu on form view.
This module helps you to filter out useless files leaving only attachments you really need.

Main features

  • Block spam or any unwanted files
  • Blacklist attachments based on file name patterns (e.g. *.exe, spamfile___.*)
  • Blacklist attachments based on file size (e.g. "all *.pdf files > 25 MB")
  • Blacklist attachments based on file contents (in case same file can have different names)
  • Blacklisted attachments will be hidden and can be restored later or deleted completely
  • Hide duplicate attachments in "Attachment(s)" menu - display each attachment only once Click to get Pro Version!
  • Blacklist attachment by contents directly from "Attachment(s)" menu - no need for "Technical Settings" access rights! Click to get Pro Version!
  • Manage attachment blacklist by contents directly from "Attachment(s)" menu - no need for "Technical Settings" access rights! Click to get Pro Version!
  • Show notification if uploaded attachment is blacklisted or already uploaded to same record Click to get Pro Version!
  • User must be added to "Manage Attachment Blacklist" to be able to blacklist attachments
